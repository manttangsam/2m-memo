# Infinite Note Android Prototype

This is a native Android/Kotlin prototype of the infinite canvas note app.

## Included

- Infinite canvas with pan and pinch zoom
- Pen drawing and eraser
- Basic mind map nodes
- Add child node
- Fold/unfold selected node
- Auto layout
- Undo snapshot support

## Open

Open this folder in Android Studio:

`outputs/android-infinite-note`

Android Studio will download the Android Gradle Plugin and Kotlin plugin if needed.

## Current limitation

This Codex environment does not have Java, Gradle, or Android SDK configured, so the APK was not built here.
