# 2M Memo Usage Guide

## Main Toolbar

- `✎`: Pen mode
- `⌫`: Eraser mode
- `↔`: Move mode
- `▣`: Save box
- `?`: Open the in-app help document
- `◎`: Mind-map mode
- `⟳`: Rotation tools
- `🔓 / 🔒`: Toggle rotation lock
- `↶`: Undo
- `☷`: Auto-layout mind map
- `⌖`: Return to canvas center
- `▦− / ▦+`: Make grid lighter or darker

## Handwriting

Use pen mode to write on the infinite canvas. The pen submenu provides pen type, thickness, and color controls.

## Eraser

Use eraser mode to erase handwriting strokes. In eraser mode, tapping a mind-map box deletes that box and its children. The `🧹` button clears all handwriting strokes.

## Mind Map

Mind-map mode supports center topics, child nodes, sibling nodes, folding, arrows, themes, templates, box resizing, handwriting input, and typed input.

Auto-layout only runs when the `☷` auto-layout button is pressed.

## Saving

Use `▣` to save a memo by folder and memo name. Saved memos can be loaded from the same save box.

## Navigation

The lower-right mini map shows memo content and the current viewport. The center crosshair marks the canvas origin.
