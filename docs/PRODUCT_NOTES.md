# Product Notes

## Product Idea

2M Memo combines memo writing and mind mapping on an infinite canvas. It is designed first for mobile pen input, then for PC editing through a web editor.

## Storage Principle

The service should act as an editor, not as the owner of user data. User memo files should remain in the user's own account or storage provider.

## Planned Web Editor

- Host the web editor on Netlify.
- Open and save local `.2memo` files first.
- Add Google Drive or OneDrive file picker integration later.
- Keep the file format compatible with the Android app.
